# Student Performance Prediction System

## 📌 Overview
This project is a Machine Learning-based system designed to predict student academic performance (Pass/Fail).  
It analyzes key factors such as study hours, attendance, previous marks, assignments, and internal marks to provide accurate predictions.

---

## 🎯 Objective
- Identify students at risk of failing  
- Help educators take preventive actions  
- Improve academic performance using data-driven insights  

---

## 🚀 Features
- Predicts student performance (Pass/Fail)  
- Uses multiple input features:
  - Study Hours  
  - Attendance  
  - Previous Marks  
  - Assignments  
  - Internal Marks  
- Interactive GUI application using Tkinter  
- Data visualization using graphs (count plot, scatter plot, box plot)  
- Accurate predictions using Machine Learning  

---

## 🛠️ Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Tkinter (GUI Development)  
- Excel (Dataset)  

---

## 🤖 Machine Learning Models
- Logistic Regression  
- Decision Tree  
- Random Forest (Best Performing Model)  

---

## 📊 System Workflow
1. Data Collection  
2. Data Preprocessing  
3. Feature Selection  
4. Model Training  
5. Model Testing  
6. Prediction Output  
7. Performance Evaluation  

---

## 📊 Output
- Predicts whether a student will PASS or FAIL  
- Displays graphs for better understanding of student performance
